这样在本地开发的时候，开发者为了搭建一个完整的开发环境，就需要装一个 nginx。团队人多了以后，nginx 的规则同步是个让人头疼的事情，特别是配置越来越复杂的时候，每个新同事进来都要折腾一下。

node proxy 代替 nginx
